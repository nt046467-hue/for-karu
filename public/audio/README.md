# Audio files

Drop your audio files into this folder:

- `background.mp3` — soft instrumental background track (royalty-free,
  search YouTube Audio Library / Artlist / Epidemic Sound for
  "emotional piano proposal"). Should loop cleanly.

- `chime.mp3` — short sparkle/chime sound, ~1 second, plays when the
  ring box opens.

## If files are missing

The experience will continue silently — Howler fails gracefully on 404.
But for the real moment, you absolutely want these in place.
